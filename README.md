# 外用類固醇藥膏查詢工具

這是一個離線可用的單頁查詢工具。商品名與成分資料可由食藥署官方開放資料更新，外用類固醇強度則由 `data/steroid-potency.json` 的人工校對表補足。

資料匯入時會排除鼻用噴液、眼用、耳用、口內、痔瘡、陰道等非一般皮膚外用製劑。

## 使用方式

直接開啟 `index.html` 即可查詢。頁面會讀取 `data/products.js` 與 `data/steroid-potency.js`，因此不需要啟動本機伺服器。

## 更新官方資料

在此資料夾執行：

```powershell
node .\scripts\update-data.mjs
```

腳本會下載並合併：

- 食藥署 `9123 未註銷藥品許可證資料集`
- 食藥署 `9121 藥品詳細處方成分資料集`

輸出：

- `data/products.json`
- `data/products.js`

## 分級原則

頁面以美國外用類固醇 7 級分類為主，但畫面用台灣使用者較直覺的中文與數字呈現：

- 美國第 1 級：非常強，最強
- 美國第 2 級：強，強效
- 美國第 3 級：中，中偏強
- 美國第 4 級：中，中等
- 美國第 5 級：中，中偏弱
- 美國第 6 級：弱，弱效
- 美國第 7 級：弱，最弱

本工具只整理藥品資訊與用藥安全提醒，不取代醫師或藥師判斷。
